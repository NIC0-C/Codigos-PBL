# VGA Controller — Digital Design (Verilog)

Academic project developed as part of a PBL in Digital Circuits.

## Description
This project implements a VGA controller using Verilog, including horizontal and vertical synchronization signals, counters, and color control logic.

Due to file size limitations, the full project is available as a compressed file.
